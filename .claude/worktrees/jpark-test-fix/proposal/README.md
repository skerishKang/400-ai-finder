# AI파인더 제안서 자료

이 폴더는 AI파인더의 사업계획서, 기관 제안서, PoC 제안서 초안을 보관한다.

## 목적

`presentation/` 폴더가 발표와 소개를 위한 자료라면, `proposal/` 폴더는 실제 제안과 계약 전 검토를 위한 문서다.

## 문서 후보

- ai-finder-business-plan-outline.md
- ai-finder-public-sector-proposal-outline.md
- ai-finder-poc-plan.md

## 작성 원칙

- 문제, 해결, 도입효과, 구현범위, 운영방식, 예산구조를 명확히 쓴다.
- 기관 대상 문서는 보안, 개인정보, 유지관리, 출처 기반 답변 원칙을 반드시 포함한다.
- 기술 문서는 과장하지 않고 MVP와 장기 확장 범위를 구분한다.
