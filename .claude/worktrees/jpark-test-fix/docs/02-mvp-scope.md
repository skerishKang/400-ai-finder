# MVP 범위

## MVP 목표

초기 MVP는 브라우저 확장프로그램 없이 특정 홈페이지를 수집하고, 사용자의 자연어 질문에 대해 관련 페이지와 문서를 찾아주는 것을 목표로 한다.

## 포함 기능

- 홈페이지 URL 기반 수집
- sitemap.xml 분석
- 주요 메뉴 구조 분석
- 게시판과 공지사항 수집
- 첨부파일 링크 수집
- PDF, HWP/HWPX, DOCX, XLSX 문서 분석
- 자연어 질문 기반 검색
- 관련 페이지 추천
- 신청 절차 요약
- 제출서류, 기한, 담당자 추출
- 답변 출처 제공

## 제외 기능

- 사용자의 현재 화면 실시간 인식
- 브라우저 확장프로그램
- 자동 클릭
- 자동 입력
- 로그인 후 개인화 화면 분석
- 결제, 신청, 제출 대행
- 모바일 앱 화면 분석

## MVP 성공 기준

사용자가 특정 홈페이지에 대해 질문했을 때 다음 정보를 안정적으로 제공할 수 있어야 한다.

1. 관련 페이지 링크
2. 관련 공지 또는 게시글
3. 관련 첨부파일
4. 사용자가 해야 할 절차
5. 제출서류
6. 기한
7. 담당자 또는 문의처
8. 답변 근거
