# 샘플 사용자 질문

## 지원사업 탐색

- 중소기업 지원사업 공고 어디 있어?
- 교육비 지원받는 공고 찾아줘.
- 이번 달 신청 가능한 사업 알려줘.
- 신청 대상이 되는지 확인하고 싶어.

## 문서와 양식 탐색

- 신청서 양식 어디서 받아?
- 제출서류가 뭐야?
- 운영지침 파일 찾아줘.
- 첨부파일 중에서 꼭 봐야 하는 문서가 뭐야?

## 홈페이지 길 찾기

- 사업공고는 어디로 들어가야 해?
- 자료실 메뉴가 어디 있어?
- 이 홈페이지에서 교육 신청은 어디서 해?
- 공지사항과 사업공고 중 어디를 봐야 해?

## 절차 확인

- 신청하려면 순서가 어떻게 돼?
- 먼저 무엇을 준비해야 해?
- 온라인으로 신청하는 건지 이메일로 제출하는 건지 알려줘.
