# AI파인더 Master Brief

## 1. 한 줄 정의

AI파인더는 복잡한 기관·기업·대학 홈페이지를 사용자의 자연어 질문에 맞춰 찾아주는 AI 홈페이지 내비게이터다.

## 2. 문제 인식

많은 홈페이지는 사용자의 목적이 아니라 기관의 조직 구조와 행정 용어를 기준으로 설계되어 있다. 사용자는 정확한 메뉴명, 사업명, 공고명, 문서명을 모르면 원하는 정보를 찾기 어렵다.

특히 공공기관, 대학, 산학협력단, 기업 지원사업 홈페이지에서는 정보가 여러 위치에 흩어져 있다.

- 공지사항
- 사업공고
- 자료실
- 신청안내
- 첨부문서
- 문의 안내

사용자는 정보를 못 찾고, 담당자는 반복 문의에 대응해야 한다.

## 3. 해결 방향

AI파인더는 홈페이지를 단순 검색하는 것이 아니라, 홈페이지 구조와 문서를 함께 분석해 사용자의 목적에 맞는 경로를 안내한다.

사용자는 이렇게 질문할 수 있다.

- 신청서 어디 있어?
- 지원사업 공고 어디서 봐?
- 제출서류가 뭐야?
- 신청하려면 어디를 눌러야 해?
- 이 홈페이지에서 교육 신청은 어디서 해?

AI파인더는 관련 페이지, 첨부문서, 신청 절차, 제출서류, 기한, 문의 정보를 근거와 함께 정리한다.

## 4. 핵심 기능

1. 홈페이지 URL 기반 수집
2. 메뉴 구조 분석
3. 공지사항과 게시판 수집
4. PDF, HWP/HWPX, DOCX, XLSX 등 첨부문서 분석
5. 사용자 질문 의도 파악
6. 관련 문서와 페이지 검색
7. 신청 절차와 제출서류 요약
8. 출처 링크와 문서 근거 제공

## 5. 차별점

기존 챗봇은 주로 FAQ를 기준으로 정해진 답변을 제공한다. 반면 AI파인더는 홈페이지 자체를 분석 대상으로 삼는다.

| 구분 | 기존 FAQ 챗봇 | AI파인더 |
|---|---|---|
| 기준 | 사전 입력된 FAQ | 홈페이지 구조와 문서 |
| 답변 방식 | 정해진 문답 | 사용자 목적 기반 탐색 |
| 문서 처리 | 제한적 | 첨부문서까지 분석 |
| 안내 방식 | 설명 중심 | 경로, 링크, 다음 행동 중심 |
| 근거 제시 | 부족한 경우 많음 | URL, 문서명, 파일명 제시 |

## 6. 초기 MVP

초기 MVP는 브라우저 확장프로그램 없이 작동한다.

- 단일 URL 분석
- 홈페이지 주요 메뉴 탐색
- 문서와 링크 수집
- 검색 가능한 인덱스 생성
- 사용자 질문에 대한 근거 기반 답변

## 7. 장기 확장

1. 기관별 맞춤 AI 홈페이지 파인더
2. 홈페이지 삽입형 AI 안내 위젯
3. 브라우저 확장프로그램 기반 화면 동행형 안내
4. 관리자용 수집 상태 대시보드
5. 내부망 또는 로컬 모델 기반 보안형 구축

## 8. 핵심 메시지

AI파인더는 홈페이지를 새로 만드는 서비스가 아니다. 기존 홈페이지 위에 AI 탐색 레이어를 얹어, 사용자가 원하는 정보를 더 쉽게 찾게 만드는 서비스다.
