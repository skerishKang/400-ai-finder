400-ai-finder / AI파인더

복잡한 홈페이지를
사람 말로 찾아주는
AI 내비게이터

AI파인더는 기관, 공공기관, 대학, 기업 지원사업 홈페이지에 흩어진 메뉴·공지·자료실·첨부문서를 분석해 사용자가 원하는 페이지와 해야 할 일을 자연어로 안내합니다.

홈페이지 탐색 문서 기반 답변 바로가기 안내 근거 표시

왜 필요한가

많은 홈페이지는 사용자의 목적보다 기관의 조직 구조와 행정 용어를 기준으로 구성되어 있습니다.

  • 공지사항, 사업공고, 자료실에 정보가 분산됩니다.
  • 중요한 내용은 PDF, HWP, DOCX, XLSX 안에 들어 있습니다.
  • 사용자는 정확한 메뉴명과 문서명을 알아야 합니다.
  • 담당자는 이미 공개된 정보에 대해 반복 문의를 받습니다.

어떻게 해결하나

“신청서 어디 있어?”처럼 물으면, AI파인더가 관련 페이지와 문서를 찾아줍니다.

AI파인더는 홈페이지 구조, 게시판, 첨부문서, 링크 관계를 분석해 사용자의 목적에 맞는 경로를 안내합니다.

사용 예시

지원사업 찾기

“중소기업 교육비 지원 공고 어디 있어?”

→ 관련 공고, 신청안내, 첨부파일을 함께 제시합니다.

제출서류 확인

“이 사업 신청하려면 어떤 서류가 필요해?”

→ 공고문과 첨부문서에서 제출서류를 찾아 요약합니다.

홈페이지 길찾기

“교육 신청은 어디로 들어가야 해?”

→ 메뉴 경로와 바로가기를 안내합니다.

제품 구조

1. 홈페이지 URL 수집
2. 메뉴·게시판·링크 분석
3. 첨부문서 파싱
4. 검색 인덱스 생성
5. 사용자 질문 이해
6. 근거 기반 답변 제공

초기 MVP

  • 단일 URL 분석
  • 페이지 제목, 본문, 링크 추출
  • 내부 링크와 외부 링크 구분
  • PDF, HWP/HWPX, DOCX, XLSX 첨부파일 분류
  • 검색과 답변 생성을 위한 데이터 구조화

도입 방식

  • 기존 홈페이지를 전면 개편하지 않습니다.
  • 공개 홈페이지를 먼저 분석합니다.
  • 기관별 검색 인덱스를 만듭니다.
  • 필요하면 홈페이지 삽입형 위젯으로 확장합니다.
  • 보안 요구가 있으면 로컬 모델 또는 내부망 운영을 검토합니다.

핵심 메시지

AI파인더는 홈페이지를 새로 만드는 서비스가 아닙니다. 기존 홈페이지 위에 AI 탐색 레이어를 얹어, 사용자가 원하는 정보를 더 쉽게 찾게 만드는 서비스입니다.
