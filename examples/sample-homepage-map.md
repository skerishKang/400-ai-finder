# 샘플 홈페이지 지도

```text
기관 홈페이지
├─ 기관소개
├─ 알림마당
│  ├─ 공지사항
│  ├─ 사업공고
│  └─ 보도자료
├─ 지원사업
│  ├─ 사업안내
│  ├─ 신청안내
│  └─ 선정결과
├─ 자료실
│  ├─ 신청서 양식
│  ├─ 운영지침
│  └─ 자주 묻는 질문
└─ 문의처
```

## 해석

- 지원사업, 보조금, 교육비, 신청 방법 관련 질문은 `지원사업`과 `알림마당 > 사업공고`를 우선 탐색한다.
- 신청서, 양식, 제출서류 관련 질문은 `자료실`과 공고 첨부파일을 함께 확인한다.
- 문의 관련 질문은 공고문 하단과 `문의처` 메뉴를 확인한다.
