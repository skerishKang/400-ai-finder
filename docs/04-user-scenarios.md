# 사용자 시나리오

## 시나리오 1. 지원사업 공고 찾기

사용자 질문:

> 중소기업 직원 교육비 지원받는 공고 어디 있어?

AI파인더 답변 목표:

- 관련 사업공고 페이지를 찾는다.
- 공고 제목과 게시일을 보여준다.
- 신청 대상, 기한, 제출서류를 요약한다.
- 신청서 파일 링크를 제공한다.
- 담당자 또는 문의처를 제공한다.

## 시나리오 2. 신청서 양식 찾기

사용자 질문:

> 신청서 양식 어디서 받아?

AI파인더 답변 목표:

- 관련 자료실 또는 공고 첨부파일을 찾는다.
- 양식 파일명을 보여준다.
- 어느 공고에 연결된 양식인지 설명한다.
- 다운로드 링크를 제공한다.

## 시나리오 3. 홈페이지 메뉴 설명

사용자 질문:

> 이 홈페이지에서 사업공고는 어디로 들어가야 해?

AI파인더 답변 목표:

- 홈페이지 메뉴 경로를 단계별로 안내한다.
- 관련 메뉴 후보를 함께 제시한다.
- 가장 가능성 높은 바로가기 링크를 제공한다.

## 시나리오 4. 제출서류 확인

사용자 질문:

> 이 사업 신청하려면 어떤 서류가 필요해?

AI파인더 답변 목표:

- 공고문과 첨부파일에서 제출서류 목록을 추출한다.
- 필수 서류와 선택 서류를 구분한다.
- 원문 근거가 불확실한 항목은 확인 필요로 표시한다.

## 시나리오 5. 담당자 문의처 찾기

사용자 질문:

> 이거 담당자 연락처 찾아줘.

AI파인더 답변 목표:

- 공고문 본문과 첨부파일에서 담당자 정보를 찾는다.
- 부서명, 이름, 전화번호, 이메일을 가능한 범위에서 제공한다.
- 개인정보성 정보는 원문에 공개된 범위만 제공한다.
