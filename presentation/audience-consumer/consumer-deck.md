# 소비자 대상 발표자료: AI파인더 소개

## 1. 제목

AI파인더: 복잡한 홈페이지에서 원하는 정보를 대신 찾아주는 도우미

## 2. 홈페이지에서 길을 잃는 문제

많은 홈페이지는 메뉴가 복잡하다. 원하는 정보가 어디 있는지 알기 어렵고, 공지사항과 자료실과 신청안내를 계속 오가야 한다.

특히 이런 상황에서 어렵다.

- 신청서가 어디 있는지 모를 때
- 공고문은 찾았지만 제출서류를 모르겠을 때
- 자료실에 파일이 너무 많을 때
- 메뉴 이름이 어려울 때
- 모바일에서 메뉴가 잘 보이지 않을 때

## 3. AI파인더에게 그냥 물어보기

AI파인더는 사용자가 정확한 메뉴명을 몰라도 질문을 이해한다.

예시 질문:

- 신청서 어디 있어?
- 지원사업 공고 찾아줘.
- 제출서류가 뭐야?
- 어디로 들어가야 해?
- 이 홈페이지에서 교육 신청은 어떻게 해?

## 4. 바로가기 안내

AI파인더는 단순히 설명만 하지 않는다. 관련 페이지와 문서로 이동할 수 있는 바로가기를 함께 제공한다.

예시:

- 사업공고 페이지
- 신청안내 페이지
- 신청서 양식
- 운영지침 문서
- 자주 묻는 질문

## 5. 해야 할 일 정리

복잡한 공고문도 사용자가 해야 할 일 중심으로 정리한다.

예시:

1. 공고문 확인
2. 신청 대상 확인
3. 신청서 다운로드
4. 제출서류 준비
5. 안내된 방식으로 제출

## 6. 문서 안의 핵심 정보 찾기

많은 홈페이지는 중요한 정보가 본문이 아니라 첨부파일 안에 있다. AI파인더는 첨부문서까지 확인해 필요한 정보를 찾는 것을 목표로 한다.

찾을 수 있는 정보:

- 신청기한
- 제출서류
- 신청대상
- 신청방법
- 양식 파일
- 문의 안내

## 7. 누가 쓰면 좋은가

AI파인더는 홈페이지를 자주 이용하지만 메뉴 구조가 익숙하지 않은 사람에게 유용하다.

- 학생
- 학부모
- 민원인
- 소상공인
- 중소기업 담당자
- 교육 신청자
- 지원사업 신청자
- 일반 시민

## 8. 기존 검색과 다른 점

일반 검색창은 정확한 단어를 입력해야 한다. AI파인더는 사용자의 말뜻을 이해하고, 관련 페이지와 문서를 함께 찾아준다.

| 일반 검색 | AI파인더 |
|---|---|
| 키워드 중심 | 질문 중심 |
| 결과 목록 제공 | 해야 할 일 안내 |
| 문서를 직접 읽어야 함 | 핵심 내용 요약 |
| 메뉴를 직접 찾아야 함 | 메뉴 경로 안내 |

## 9. 핵심 메시지

AI파인더는 어려운 홈페이지를 쉽게 쓰게 해주는 도우미다.

정확한 메뉴명을 몰라도 괜찮다. 필요한 정보를 사람 말로 물어보면, AI파인더가 관련 페이지와 문서를 찾아준다.
