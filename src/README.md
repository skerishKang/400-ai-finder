# src

이 폴더는 AI파인더 구현 코드를 위한 공간입니다.

- agent/: 시민 행동 계획 및 퀘스트 기반 액션 플랜 생성
- analytics/: 질문 로깅 및 반복 질문 분석
- answer/: 답변 구성, 상태 관리 및 URL 검증
- config/: 설정 및 상수 관리
- crawler/: URL 수집, 분류, 사이트맵 파싱 및 경로 필터링
- demo/: 제어된 라이브 데모, 대화 기록 및 UX 실행 도구
- diagnostics/: 사이트 진단 및 상태 점검
- fetch/: HTTP 요청, Firecrawl 통합, 데이터 정제
- indexer/: 문서 색인 생성 및 문서 보강
- llm/: LLM 제공자 추상화 및 라우팅 (OpenAI, Mock, Stub)
- observability/: 이벤트 로깅 및 관측 가능성
- pipeline/: 파이프라인 실행 및 스모크 리포트 생성
- search/: 키워드 검색, 질의 재작성 및 소스 매칭
- site_profiles/: 사이트별 프로필 관리
- strategy/: 전략 라우팅
- web/: 정적 파일 서버, 모바일/관리자 데모
